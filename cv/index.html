— Proyectos

Nata100 (2020)
Diseño e implementación de una aplicación web destinada a compartir marcas entre nadadores y generar retroalimentación de parte de entrenadores. Proyecto realizado en el framework Django. Realizado para los cursos EL5003 Introducción al Taller de Diseño y EL5004 Taller de Diseño, Universidad de Chile.
Theremin digital inalámbrico (2020)
Programación de un instrumento musical basado en un theremin, usando sensores de proximidad, comunicación Bluetooth y microcontroladores Arduino. Realizado para el curso EL5202 Laboratorio de Sistemas Digitales, Universidad de Chile.
Transmisión de imágenes mediante portadoras audibles (2018-2019)
Implementación de un sistema de comunicación paquetizada de imágenes a través de ondas de audio. Realizado para el curso EL4005 Principios de Comunicaciones y extendido para el curso EL5207 Laboratorio de Tecnologías de Información y Comunicaciones, Universidad de Chile.
Piano digital (2018)
Construcción y programación de un piano digital a partir de un teclado PS/2 usando el lenguaje Verilog. Realizado para el curso EL4002 Sistemas Digitales, Universidad de Chile.
Transmisor FM (2017)
Diseño y construcción de un transmisor de ondas FM. Realizado para el curso EL3004 Circuitos Electrónicos y Analógicos, Universidad de Chile.
Altavoz (2017)
Diseño y construcción de un altavoz electrónico dinámico. Realizado para el curso EL3003 Laboratorio de Ingeniería Eléctrica.
Robot bípedo (2016)
Diseño, construcción y programación de un robot bípedo. Realizado para el curso EI2001-3 Mecatrónica, Universidad de Chile.
